# Sales Performance Analysis System

## Overview
This project analyzes sales data using Python, Pandas, NumPy, and Matplotlib.

## Features
- Data Cleaning
- Sales Analysis
- Top Selling Products
- Monthly Sales
- Category-wise Sales
- Bar Chart
- Line Chart
- Pie Chart
- Report Generation

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib

## Run

```bash
pip install -r requirements.txt
python main.py
```
